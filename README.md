# TikShopping - Catálogo SAAS con integración TikTok

**Curso:** Taller de Desarrollo Web  
**Equipo:** Grupo 1  
**Año:** 2026

## Descripción
TikShopping es una plataforma SAAS (Software as a Service) que permite a pequeños negocios crear y publicar su catálogo de productos de forma sencilla, con la posibilidad de integrarse en videos de TikTok para mostrar el catálogo en TikShopping.

## Características principales
- 🏠 Página de inicio con lista de negocios destacados.
- 📦 Catálogo de productos por negocio.
- 📝 Formulario de sugerencias para clientes.
- 📄 Libro de reclamaciones.
- 📍 Contáctenos con mapa de ubicación.
- 🎥 Link/Función de TikShopping incrustados en videos de TikTok.
- 📱 Diseño completamente responsivo.
- 🔐 Simulación de registro/login. 

## Tecnologías utilizadas
- HTML5
- CSS
- JavaScript 
- GitHub Pages
  
## Estructura del proyecto

| Archivo/Carpeta | Descripción |
|----------------|-------------|
| `├── index.html` | Página de inicio con negocios destacados |
| `├── catalogo.html` | Catálogo de productos de un negocio |
| `├── sugerencias.html` | Formulario de sugerencias |
| `├── reclamaciones.html` | Libro de reclamaciones |
| `├── contactenos.html` | Formulario de contacto y mapa |
| `├── login.html` | Simulación de registro/login |
| `├── css/` | Carpeta de estilos |
| `│   └── styles.css` | Estilos responsivos |
| `├── js/` | Carpeta de scripts |
| `│   └── main.js` | Validaciones y lógica |
| `└── assets/imagenes/` | Imágenes del proyecto |
    
## Integrantes (roles)
- **Persona 1** - Formulario de sugerencias + Inicio
- **Persona 2** - Libro de reclamaciones + Catálogo
- **Persona 3** - Contáctenos/Ubícanos + Videos TikTok
- **Persona 4** - Layout global, responsivo, integración
